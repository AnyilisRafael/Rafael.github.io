Política de Privacidad

Esta política de privacidad aplica a todas las aplicaciones desarrolladas bajo este proyecto. A continuación se detalla cómo se recopilan, usan y protegen los datos dentro de las aplicaciones.

Datos que recopilamos

  • Datos de uso local: puntuaciones, progreso y estadísticas dentro del juego o aplicación.
  • Datos persistentes: récords guardados, configuraciones del usuario y estado de la aplicación.

Nota: Ninguna de las aplicaciones recopila datos personales sensibles, no utilizan identificadores únicos del dispositivo.

Uso de los datos

  • Permitir que el usuario continúe su progreso en diferentes sesiones.
  • Mejorar la experiencia mediante la conservación de récords y configuraciones.
  • Optimizar el rendimiento de forma local en el dispositivo.

Servicios de terceros

Las aplicaciones podrían contener anuncios gestionados por servicios de terceros. En ese caso, dichos anuncios pueden usar identificadores no personales para mostrar contenido relevante. No se recopilan datos sensibles ni se comparten con fines distintos a la funcionalidad de la aplicación.

Seguridad

Los datos se almacenan únicamente en el dispositivo del usuario, utilizando los mecanismos de seguridad provistos por el sistema operativo. No se transmiten ni se sincronizan con servidores externos.

Derechos del usuario

El usuario puede eliminar manualmente los datos guardados desinstalando la aplicación o restableciendo su progreso desde las opciones internas.

Actualizaciones de esta política

Podemos actualizar esta política de privacidad para reflejar mejoras en las aplicaciones o cambios legales. La versión más reciente siempre estará disponible en el mismo enlace.
